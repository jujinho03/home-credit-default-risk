# 포트폴리오 프로젝트 설명

## 프로젝트명

Home Credit 채무불이행 위험 분석

## 한 줄 소개

307,511건의 대출 신청 데이터를 검수하고 5개 분류모델을 비교해, 위험 상위 10%에서 전체 대비 3.28배의 부도율을 확인한 학부 머신러닝 프로젝트입니다.

## 문제와 접근

부도 고객이 전체의 8.07%인 불균형 데이터에서 단순 정확도 대신 ROC-AUC와 위험구간별 부도율을 사용했습니다. 데이터 누수를 막기 위해 학습·검증 데이터를 먼저 나눈 뒤 학습 데이터에서만 결측치 대체값과 표준화 기준을 계산했습니다.

## 수행 내용

- 결측치·이상코드·중복·학습/검증 데이터 중첩 점검
- 13개 파생변수 생성과 층화 학습/검증 분할
- Logistic Regression, LDA, SVM 계열 모델 비교
- 교차검증, 변수 중요도, 임계값과 위험구간 분석
- 잘못 비교한 변수 중요도를 발견하고 기준을 맞춰 재분석
- 재현 가능한 노트북, 결과표와 분석 보고서 작성

## 핵심 결과

- 기준모델 L2 Logistic Regression의 검증 ROC-AUC 0.7504
- 위험 상위 10% 실제 부도율 26.48%
- 전체 부도율 대비 Lift 3.28배
- 희소 조직유형의 중요도 과대평가 문제 발견 및 수정

## 사용 기술

Python, pandas, NumPy, scikit-learn, SciPy, Matplotlib, JupyterLab, Git

## 직무 역량 연결

- Python을 이용한 데이터 전처리·검수
- 분류모델 학습과 평가 결과 비교
- 분석 오류 확인과 수정 근거 기록
- 재현 가능한 노트북, 결과표와 보고서 작성

위 경험은 데이터분석 연구 및 개발결과물 작성 업무에 필요한 기초 역량을 보여줍니다.

## 설명 가능한 범위

이 프로젝트는 학부 수업에서 학습한 전처리, 선형 분류모델, SVM, 교차검증과 기본 평가 지표를 적용한 결과입니다. 모델을 실제 대출 승인에 바로 사용하는 수준이 아니라, 데이터 검수와 모델 비교 과정을 재현하고 결과의 한계를 설명하는 포트폴리오입니다.
